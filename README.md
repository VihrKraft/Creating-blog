# Блог

Этот проект является основой, которую вы можете использовать для создания своего блога.

## Как запустить

Сайт можно запустить предварительно скачав файлы. Существует два способа: 
1. Скачать проект в виде архива (`Add file` => `Upload files`) и разархивировать его в нужном для вас месте.
2. Сделать `Fork` проекта и клонировать его на компьютер с помощью `GitHub Desktop`.

После проделанного зайти в корневую папку сайта в проводнике, зайти в папку `dist` и открыть файл `index.html` в браузере.

## Установка зависимостей

Для работы с проектом нужно установить [Node.js](https://nodejs.org/en/) и следующие зависимости:
```
npm install pug -D

npm install webpack -D

npm install webpack-cli webpack-dev-server -D

npm install pug-plugin --save-dev

npm install --save-dev css-loader

npm install sass-loader sass webpack --save-dev
```

## Описание файлов

`index.pug` - это основной файл проекта, который объединяет все `pug`-файлы, хранящиеся в папке `html`.

`scss/style.scss` - файл, объединяющий все `.scss` файлы.

`index.js` - содержит динамическую составляющую сайта.

`package-lock.json`, `package.json` и `webpack.config.js` нужны для коомпиляции `.pug` и `.scss` в `.html` и `.css`.

## Команды

Для работы с проектом рекомендуется использовать команды: 

`npm run start` - создает локальный сервер, с помощью которого можно динамично писать код.

`npm run dev` - создает тестовую сборку в папке `dist`.

`npm run build` - создает готовую сборку в папке `dist`.

`npm run clear` - удаляет папку `dist`.
